推荐几个Windows下的实用小工具（上）

发表于 2016-03-25   |   分类于 随笔   |     |   访问量:

在我的电脑中安装了一些在我看来能够美化Windows界面和提高工作效率的软件，如果能熟练应用这些工具，会使日常的工作变得更加方便与舒适。在这里我就分享一些给大家。

StartIsBack++

StartIsBack是一款能够让Win8或Win10的开始菜单变成类似于Win7一样的模式的小工具，具体到给Win10用的版本叫做StartIsBack++。有些人可能不理解为什么要把Win10的开始菜单也变成像Win7一样，实际上是我认为Win10开始菜单中“所有应用”都是以首字母分类，自我感觉没有Win7那样的文件夹模式使用起来方便。另一个更为重要的方面是它真的是好漂亮啊，晒图如下。

当然它有很多视觉样式可以选择，也可以自定义任务栏的颜色与透明度。我选择了如上图所示的样式是因为它把Win7的开始菜单功能与Win10的Metro界面风格完美地融合在了一起。你也可以自定义右侧所显示的快捷项目，可以看到我把控制面板、设置、命令提示符、运行这些常用功能都加了进去。

什么？你说你突然想用原生的开始菜单（比如想打开Metro应用）怎么办？不用担心，只需要用鼠标中键单击Windows图标就可以弹出原生菜单啦，是不是很酷？
这里是StartIsBack的官方网站，支持正版，可以使用PayPal购买，只要2.99美元。另有软件Start10和这个差不多，可以在淘宝的异次元正版商城购买正版，现在卖42元。

Bing每日壁纸

不知道有多少人每隔一段时间都要为换壁纸发愁，有了这个神器，你就再也不用操心壁纸的问题啦，尤其推荐给那些对壁纸喜新厌旧又懒得寻找的人。
我们知道，Bing搜索每天的背景图质量都非常高，适合作桌面壁纸（如下图）。但是从它的网站下载的图片会有水印，而且要把它设为壁纸也要费会儿功夫。

于是，这个神器出现了！它可以使你每天打开电脑时的桌面壁纸都变为高清无水印的当天的Bing背景图！只要你把它设置为开机自动启动，你就可以看到你每天的壁纸都是不一样的，而且绝对漂亮，每一天都是新的一天，都值得期待。它的界面如下图。

可以看到，它能把每天下载的壁纸存放在用户自定义的目录中，你如果觉得哪一天的壁纸特别漂亮，可以到这个目录中寻找这个图片文件。这里大概展示一些壁纸以说明这些壁纸有多么好看。

这里是Bing每日壁纸的下载地址：http://www.xdowns.com/soft/6/99/2014/Soft_116540.html
