推荐几个Windows下的实用小工具（中）

发表于 2016-09-09   |   分类于 随笔   |     |   访问量:

大家好！在今年3月我向大家推荐了两个小工具，然后就一直没有再更新，这可能是因为我太懒了……不管怎样，该来的还是要来的，今天开始终于要把剩下的神器小软件都推荐给大家了！这次推荐的是文件搜索神器Listary。

请注意：上一次推荐的下载链接中的Bing每日壁纸软件好像已经不能用了（至少是我的电脑不能用了），请点击这里下载可用的版本（记得设置开机启动哦）。

Listary

Listary是一款极为优秀的文件浏览与搜索增强工具。你是不是有过在文件资源管理器里辛辛苦苦打开一个又一个文件夹来查找文件的经历？是不是有过在开始菜单中从前往后仔细翻看来寻找一个应用程序的经历？如果你苦于这些繁琐恼人的查找工作，那么Listary就是专门来帮你解脱的！

我最常用也是基本唯一在用的Listary功能就是快速全盘搜索（不要说什么Windows有自带的搜索功能，那个搜索速度是你能忍受的？），在任何时候只需要用快捷键呼出Listary的搜索框（默认是连按两下Ctrl键），就可以进行全盘搜索，且支持拼音简写，搜索速度超级快。

此时可以按上下方向键选定项目来回车打开，也可以在选定项目后按→键打开快捷操作列表，包括一些常用动作和右键菜单选项，像是这样：

是不是很厉害？这样的话文件管理器和开始菜单都可以扔掉了呢！（夸张了呵呵）

其实Listary作为一款文件资源管理器的增强工具，配合文件管理器食用也是非常酷的。在浏览文件时直接打字就可以在当前目录下进行搜索，在任意软件的“打开”对话框下面都会有Listary的快捷搜索栏，并可以通过搜索栏的菜单选项或是快捷键转到当前已经打开的文件夹。

其他功能我就不一一介绍了，这里有一个演示Listary基本功能的视频分享给大家（我试了一下，这个视频里展示的功能并非都能在我的Listary Pro 5.00.2410中实现，比如那个输入大于号进行全盘搜索的功能和那个CTRL+O的功能。但是全盘搜索直接在任何时候用快捷键打开搜索框就可以了，CTRL+O的功能用得也不多，这都不是问题）：


Listary的基本版是免费的（支持简体中文），大家可以从它的官网下载。Listary有很多实用的增强功能和特性如 Project (项目汇聚搜索)和FuzzyNavigation (搜索当前文件夹里面的所有文件)是Listary Pro专业版专享的，免费版并不能使用。支持正版软件，可以在淘宝网的异次元正版商城购买正版Listary Pro，现在卖59元（我只是给大家提供购买软件的方法，并不是在做广告啊~）。
